# 🎵 Music Player Docker - 群辉 (Synology) 安装指南

> **最新版本** | 在线音乐播放器 - 多源搜索 / SPA 界面 / 国内音源

## 系统要求

- 群辉 NAS (DSM 7.0+) 已启用 Docker 套件（Container Manager）
- 至少 512MB 可用内存
- 至少 1GB 可用磁盘空间

---

## 安装方法

### 方式一：SSH + Docker Compose（推荐）

#### 步骤 1：SSH 登录群辉 NAS

```bash
ssh 你的用户名@群辉NAS的IP
```

> 如果 SSH 未启用：DSM → 控制面板 → 终端机和 SNMP → 启用 SSH

#### 步骤 2：创建工作目录

```bash
sudo mkdir -p /volume1/docker/music-player
cd /volume1/docker/music-player
```

#### 步骤 3：下载主程序文件

从 GitHub 下载 `main.py`：

```bash
# 尝试 GitHub 加速镜像
sudo wget -O main.py https://ghfast.top/https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/fnos/main.py
```

如果上面失败，试试其他镜像：

```bash
# 镜像 1
sudo wget -O main.py https://gh-proxy.com/https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/fnos/main.py

# 镜像 2
sudo wget -O main.py https://gitdl.cn/https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/fnos/main.py
```

下载 Dockerfile 和 docker-compose.yml：

```bash
sudo wget -O Dockerfile https://ghfast.top/https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/synology/Dockerfile
sudo wget -O docker-compose.yml https://ghfast.top/https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/synology/docker-compose.yml
```

#### 步骤 4：构建镜像

```bash
sudo docker compose build --no-cache
```

⏱️ 首次构建约需 3-5 分钟。

#### 步骤 5：启动容器

```bash
sudo docker compose up -d
```

#### 步骤 6：验证

```bash
sudo docker compose ps
sudo docker compose logs -f
curl http://localhost:8080/
```

访问 `http://群辉NAS的IP:8080/player`

### 方式二：Container Manager 图形界面

#### 步骤 1：下载项目文件

在能访问 GitHub 的环境下载以下文件：
- `fnos/main.py`（从 GitHub 仓库下载）
- `synology/Dockerfile`
- `synology/docker-compose.yml`

#### 步骤 2：上传到群辉

1. 登录 DSM → **文件管理**
2. 创建目录：`/volume1/docker/music-player/`
3. 上传 `main.py`、`Dockerfile`、`docker-compose.yml` 到该目录

#### 步骤 3：通过 SSH 构建

```bash
cd /volume1/docker/music-player
sudo docker compose build --no-cache
sudo docker compose up -d
```

## 版本管理

### 使用最新版本（默认）

上述方式默认构建最新版本。每次重新构建即可获取最新代码。

### 指定版本构建

如果需要指定版本（如 `v0.3.1`）：

```bash
cd /volume1/docker/music-player

# 下载指定版本的 main.py
sudo wget -O main.py https://raw.githubusercontent.com/nbqcw123/music-player-docker/v0.3.1/fnos/main.py

# 重新构建
sudo docker compose build --no-cache
sudo docker compose up -d
```

> 注意：将 `v0.3.1` 替换为你需要的版本 tag。

## 常见问题

### 容器无法启动
```bash
sudo docker compose logs -f
```

### 搜索无结果
YouTube 可能受网络影响，切换网易云/B站/QQ/酷狗。

### 端口冲突
修改 `docker-compose.yml` 中的 `ports`：
```yaml
ports:
  - "9090:8080"   # 改用 9090 端口
```
然后访问 `http://NAS的IP:9090/player`

### Container Manager 中找不到 Docker
DSM 7.0+ 中 Docker 改名为 **Container Manager**，在套件中心搜索安装。
