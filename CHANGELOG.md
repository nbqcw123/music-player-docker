# v0.3.3 (2026-06-04)

## B站播放修复 + 搜索预检测

### 修复
- 🎬 **B站音频流** — 改用B站API直接获取DASH音频流，替代yt-dlp（解决防盗链导致的播放失败）
- 🔗 **Proxy动态Referer** — 根据URL域名自动设置对应Referer（B站/网易/QQ/酷狗各自对应）
- ✅ **Check动态Referer** — 同上，检测链接可用性时使用正确的Referer

### 新增
- 🔍 **搜索预检测** — 搜索后自动并发检测前30首链接可用性，只返回能播放的歌曲
- 📊 **过滤日志** — 控制台输出不可播放歌曲信息和检测统计
