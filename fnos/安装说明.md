# 🎵 Music Player Docker - 飞牛 NAS 安装指南

> **版本：v0.1.0-dev** | 在线音乐播放器

## 功能特性

- 🔍 多源搜索：YouTube / Bilibili / 网易云音乐 / SoundCloud / FMA
- 🎧 支持无损格式 (FLAC/APE/WAV) + MP3/AAC/OGG/M4A
- 📱 深色科技风自适应网页界面
- 🔥 各平台热门音乐推荐
- 📋 侧边栏播放列表管理

## 系统要求

- 飞牛 NAS (FnOS) 已启用 Docker 功能
- 至少 512MB 可用内存
- 至少 1GB 可用磁盘空间

---

## 安装方法

由于 Docker Hub 和 GitHub 在部分网络环境下不稳定，提供 **三种安装方式**，按顺序尝试：

---

### 方式一：SSH + 手动下载文件（最可靠）

#### 步骤 1：SSH 登录飞牛 NAS

```bash
ssh root@飞牛NAS的IP
```

#### 步骤 2：创建工作目录

```bash
mkdir -p /vol1/docker/music-player
cd /vol1/docker/music-player
```

#### 步骤 3：创建 Dockerfile

```bash
cat > Dockerfile << 'DOCKERFILE_END'
FROM python:3.11-slim

ENV PYTHONUNBUFFERED=1 \
    PYTHONDONTWRITEBYTECODE=1 \
    PIP_NO_CACHE_DIR=1 \
    TZ=Asia/Shanghai

RUN apt-get update && apt-get install -y --no-install-recommends \
    ffmpeg \
    curl \
    && rm -rf /var/lib/apt/lists/*

WORKDIR /app

RUN pip install --no-cache-dir \
    fastapi==0.111.0 \
    uvicorn[standard]==0.30.1 \
    aiohttp==3.9.5 \
    yt-dlp==2024.5.27 \
    python-multipart==0.0.9 \
    pydantic==2.7.4 \
    lxml==5.2.2 \
    beautifulsoup4==4.12.3 \
    feedparser==6.0.11 \
    aiodns==3.2.0 \
    Brotli==1.1.0 \
    certifi==2024.6.2

COPY main.py .

RUN mkdir -p /tmp/music_cache

EXPOSE 8080

CMD ["python", "main.py"]
DOCKERFILE_END
```

#### 步骤 4：下载主程序

由于 GitHub 访问不稳定，使用国内镜像下载：

```bash
# 尝试 GitHub 加速镜像
wget -O main.py https://ghfast.top/https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/backend/main.py
```

如果上面失败，尝试其他镜像：

```bash
# 镜像 1
wget -O main.py https://gh-proxy.com/https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/backend/main.py

# 镜像 2
wget -O main.py https://gitdl.cn/https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/backend/main.py

# 镜像 3
wget -O main.py https://gh.idayer.com/https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/backend/main.py
```

#### 步骤 5：构建镜像

```bash
docker build -t music-player:dev .
```

⏱️ 首次构建约需 3-5 分钟。

#### 步骤 6：运行容器

```bash
docker run -d \
  --name music-player \
  -p 8080:8080 \
  -e TZ=Asia/Shanghai \
  -v music_cache:/tmp/music_cache \
  --restart unless-stopped \
  music-player:dev
```

#### 步骤 7：验证

```bash
docker ps | grep music-player
docker logs music-player
curl http://localhost:8080/
```

访问 `http://飞牛NAS的IP:8080/player`

### 方式三：飞牛文件管理 + Docker 界面操作

#### 步骤 1：在电脑上下载文件

在能访问 GitHub 的环境下载以下文件：
- `Dockerfile`
- `backend/main.py`

#### 步骤 2：上传到飞牛

1. 登录飞牛管理界面 → **文件管理**
2. 创建目录：`/vol1/docker/music-player/`
3. 上传 `Dockerfile` 和 `main.py` 到该目录

#### 步骤 3：SSH 构建

```bash
cd /vol1/docker/music-player
docker build -t music-player:dev .
docker run -d --name music-player -p 8080:8080 --restart unless-stopped music-player:dev
```

## 使用说明

### 浏览推荐
打开页面即可看到各平台热门音乐推荐，点击直接播放。

### 搜索音乐
顶部搜索框输入歌曲名/歌手/专辑，按回车搜索。

### 切换音源
- **🌐 全部** - 搜索所有音源
- **📺 YouTube** - YouTube
- **📹 Bilibili** - B站
- **☁️ 网易云** - 网易云音乐
- **🎧 SoundCloud** - SoundCloud

### 播放控制
底部播放栏：播放/暂停、上/下一首、进度拖拽、音量。

### 播放列表
点击右侧 **📋** 打开播放列表。

## 常见问题

### 容器无法启动
```bash
docker logs music-player
```

### 搜索无结果
YouTube/SoundCloud 可能受网络影响，切换网易云或B站。

### 端口冲突
```bash
docker run -d --name music-player -p 9090:8080 --restart unless-stopped music-player:dev
```
然后访问 `http://NAS的IP:9090/player`

### pip 构建超时
配置代理后重新构建：
```bash
docker build -t music-player:dev \
  --build-arg HTTP_PROXY=http://代理IP:端口 \
  --build-arg HTTPS_PROXY=http://代理IP:端口 \
  .
```
