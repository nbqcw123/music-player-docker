# 🎵 Music Player Docker - 飞牛 NAS 安装指南

> **版本：v0.1.0-dev** | 在线音乐播放器

## 功能特性

- 🔍 多源搜索：YouTube / Bilibili / 网易云音乐 / SoundCloud / FMA
- 🎧 支持无损格式 (FLAC/APE/WAV) 和有损格式 (MP3/AAC/OGG/M4A)
- 📱 深色科技风自适应网页界面
- 🔥 各平台热门音乐推荐列表
- 📋 侧边栏播放列表管理

## 系统要求

- 飞牛 NAS (FnOS) 已启用 Docker 功能
- 至少 512MB 可用内存
- 至少 1GB 可用磁盘空间

## 方法一：Docker Compose 导入（推荐）

### 步骤 1：登录飞牛管理界面

在浏览器中打开飞牛 NAS 的管理地址（如 `http://192.168.1.100`），登录管理员账号。

### 步骤 2：进入 Docker 管理

点击左侧菜单 **Docker** → **容器** → 点击右上角 **添加** 按钮 → 选择 **Docker Compose**

### 步骤 3：粘贴 Compose 配置

将以下内容完整粘贴到编辑器中：

```yaml
version: '3.8'

services:
  music-player:
    image: music-player:dev
    container_name: music-player
    ports:
      - "8080:8080"
    environment:
      - PORT=8080
      - CACHE_DIR=/tmp/music_cache
      - TZ=Asia/Shanghai
    volumes:
      - music_cache:/tmp/music_cache
    restart: unless-stopped
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:8080/"]
      interval: 30s
      timeout: 10s
      retries: 3

volumes:
  music_cache:
```

### 步骤 4：修改端口（可选）

如果 8080 端口已被占用，修改 `ports` 中的第一个数字：

```yaml
ports:
  - "你的端口:8080"
```

### 步骤 5：部署

点击 **部署/创建** 按钮，等待容器启动完成。

### 步骤 6：访问播放器

打开浏览器访问：

```
http://NAS的IP地址:8080/player
```

例如：`http://192.168.1.100:8080/player`

## 方法二：SSH 命令行安装

如果你的飞牛 NAS 支持 SSH 登录：

```bash
# 1. SSH 登录飞牛 NAS
ssh root@飞牛NAS的IP

# 2. 创建工作目录（在存储池中）
mkdir -p /vol1/docker/music-player

# 3. 创建 docker-compose.yml
cat > /vol1/docker/music-player/docker-compose.yml << 'EOF'
version: '3.8'

services:
  music-player:
    image: music-player:dev
    container_name: music-player
    ports:
      - "8080:8080"
    environment:
      - PORT=8080
      - CACHE_DIR=/tmp/music_cache
      - TZ=Asia/Shanghai
    volumes:
      - music_cache:/tmp/music_cache
    restart: unless-stopped
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:8080/"]
      interval: 30s
      timeout: 10s
      retries: 3

volumes:
  music_cache:
EOF

# 4. 启动
cd /vol1/docker/music-player
docker-compose up -d

# 5. 查看状态
docker-compose ps
docker-compose logs -f
```

## 使用说明

### 浏览推荐

打开页面即可看到各平台（网易云/Bilibili/YouTube/SoundCloud）的热门音乐推荐列表，点击任意歌曲开始播放。

### 搜索音乐

在顶部搜索框输入：
- **歌曲名**：如 "晴天"
- **歌手名**：如 "周杰伦"
- **专辑名**：如 "七里香"

按回车或点击搜索按钮。

### 切换音源

点击顶部音源标签：
- **🌐 全部** - 搜索所有音源
- **📺 YouTube** - 仅搜索 YouTube
- **📹 Bilibili** - 仅搜索 B站
- **☁️ 网易云** - 仅搜索网易云音乐
- **🎧 SoundCloud** - 仅搜索 SoundCloud

### 播放控制

底部播放栏支持：
- ▶/⏸ 播放/暂停
- ⏮/⏭ 上一首/下一首
- 进度条拖拽
- 音量调节

### 播放列表

点击右侧 **📋** 按钮打开播放列表侧边栏。

## 常见问题

### 0x 容器无法启动

```bash
# 检查日志
docker logs music-player

# 检查端口是否被占用
netstat -tlnp | grep 8080
```

### 搜索无结果

部分音源（YouTube、SoundCloud）可能受网络环境影响。切换到其他音源或配置代理。

### 页面无法访问

1. 检查容器是否运行：`docker ps | grep music-player`
2. 检查防火墙是否放行 8080 端口
3. 确认 NAS IP 地址正确

### 修改端口

编辑 `docker-compose.yml` 中的 `ports`:

```yaml
ports:
  - "9090:8080"   # 改为 9090
```

然后重新部署：`docker-compose up -d`

## 更新升级

```bash
cd /vol1/docker/music-player

```

```bash
cd /vol1/docker/music-player
docker-compose down -v
```
