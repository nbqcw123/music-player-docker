# 🎵 Music Player Docker - 飞牛 NAS 安装指南

> **版本：v0.1.0-dev** | 在线音乐播放器

## 功能特性

- 🔍 多源搜索：YouTube / Bilibili / 网易云音乐 / SoundCloud / FMA
- 🎧 支持无损格式 (FLAC/APE/WAV) 和有损格式 (MP3/AAC/OGG/M4A)
- 📱 深色科技风自适应网页界面
- 🔥 各平台热门音乐推荐列表
- 📋 侧边栏播放列表管理

## 系统要求

- 飞牛 NAS (FnOS) 已启用 Docker 功能
- 至少 512MB 可用内存
- 至少 1GB 可用磁盘空间
- NAS 可访问 GitHub（用于拉取源码构建镜像）

## 安装方法（SSH 命令行）

由于 Docker Hub 在国内访问不稳定，推荐通过 SSH 在飞牛 NAS 上从 GitHub 源码构建：

### 步骤 1：SSH 登录飞牛 NAS

```bash
ssh root@飞牛NAS的IP
```

### 步骤 2：克隆项目

```bash
# 选择一个有足够空间的存储池目录
mkdir -p /vol1/docker
cd /vol1/docker

# 克隆项目
git clone https://github.com/nbqcw123/music-player-docker.git
cd music-player-docker
```

如果 git clone 失败，手动下载：

```bash
mkdir -p /vol1/docker/music-player-docker
cd /vol1/docker/music-player-docker

# 下载 Dockerfile 和后端代码
wget https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/Dockerfile
wget https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/backend/requirements.txt
wget https://raw.githubusercontent.com/nbqcw123/music-player-docker/main/backend/main.py -O main.py
mkdir -p backend
mv requirements.txt main.py backend/
```

### 步骤 3：构建镜像

```bash
docker build -t music-player:dev .
```

⏱️ 首次构建约需 3-5 分钟（需要下载 Python 依赖和 ffmpeg）。

### 步骤 4：创建并启动容器

```bash
docker run -d \
  --name music-player \
  -p 8080:8080 \
  -e TZ=Asia/Shanghai \
  -v music_cache:/tmp/music_cache \
  --restart unless-stopped \
  music-player:dev
```

### 步骤 5：验证运行

```bash
# 查看容器状态
docker ps | grep music-player

# 查看日志
docker logs -f music-player

# 健康检查（应返回 {"status":"ok"}）
curl http://localhost:8080/
```

### 步骤 6：访问播放器

打开浏览器访问：

```
http://飞牛NAS的IP:8080/player
```

例如：`http://192.168.1.100:8080/player`

## Docker Compose 方式（推荐）

### 步骤 1-2：同上，克隆项目

```bash
git clone https://github.com/nbqcw123/music-player-docker.git /vol1/docker/music-player-docker
cd /vol1/docker/music-player-docker
```

### 步骤 3：使用 Compose 文件

飞牛 `docker-compose.yml` 已去除 `version` 属性，直接使用：

```bash
# 在飞牛 Docker 管理界面中导入以下 Compose 配置
# 或者命令行方式：
docker compose up -d
```

### Compose 配置内容

```yaml
services:
  music-player:
    build:
      context: .
      dockerfile: Dockerfile
    image: music-player:dev
    container_name: music-player
    ports:
      - "8080:8080"
    environment:
      - PORT=8080
      - TZ=Asia/Shanghai
    volumes:
      - music_cache:/tmp/music_cache
    restart: unless-stopped

volumes:
  music_cache:
```

## 在飞牛 Docker 管理界面中操作

### 步骤 1：上传项目文件

1. 登录飞牛管理界面 → **文件管理**
2. 在存储池中创建目录：`/vol1/docker/music-player-docker/`
3. 上传以下文件到该目录：
   - `Dockerfile`（从项目根目录）
   - `backend/` 目录（包含 `main.py` 和 `requirements.txt`）

### 步骤 2：SSH 构建镜像

```bash
cd /vol1/docker/music-player-docker
docker build -t music-player:dev .
```

### 步骤 3：在 Docker 管理界面创建容器

1. 进入 **Docker** → **容器** → **添加**
2. 选择 **使用镜像** → 找到 `music-player:dev`
3. 配置：
   - **端口映射**：`8080 → 8080`（TCP）
   - **环境变量**：`TZ=Asia/Shanghai`
   - **存储卷**：创建命名卷 `music_cache`，挂载到 `/tmp/music_cache`
   - **重启策略**：`unless-stopped`
4. 点击 **创建**

## 使用说明

### 浏览推荐
打开页面即可看到各平台（网易云/Bilibili/YouTube/SoundCloud）的热门音乐推荐列表，点击任意歌曲直接播放。

### 搜索音乐
在顶部搜索框输入歌曲名、歌手名或专辑名，按回车搜索。

### 切换音源
点击顶部音源标签：
- **🌐 全部** - 搜索所有音源
- **📺 YouTube** - 仅搜索 YouTube
- **📹 Bilibili** - 仅搜索 B站
- **☁️ 网易云** - 仅搜索网易云音乐
- **🎧 SoundCloud** - 仅搜索 SoundCloud

### 播放控制
底部播放栏支持播放/暂停、上/下一首、进度拖拽、音量调节。

### 播放列表
点击右侧 **📋** 按钮打开播放列表侧边栏。

## 常见问题

### 容器无法启动
```bash
docker logs music-player
```

### 搜索无结果
YouTube/SoundCloud 可能受网络环境影响，切换到网易云或B站。

### 端口冲突
修改端口映射，如 `9090 → 8080`，然后访问 `http://NAS的IP:9090/player`

### 构建失败（网络问题）
如果构建时 pip 安装超时，可以配置代理：
```bash
docker build -t music-player:dev \
  --build-arg HTTP_PROXY=http://代理IP:端口 \
  --build-arg HTTPS_PROXY=http://代理IP:端口 \
  .
```
